Installing Arch Linux

Alright fools. The first thing you HAVE to do is know how to RTFM. If you don't know what that means, then you've already sunk your weakass battleship into the depths of being a noob.
Since I'm not a noob (I'm definitely a noob), open the Arch Linux Wiki installation guide CLICK ME!

1) NUMBA ONE - Get the image file for ArchLinux, I just grabbed the latest one on the Arch Linux download page.
2) DEUX - Create a bootable USB with the image (You will need to be an admin for this) and type the following. Again, if you are curious as to what these ACTUALLY do I highly suggest googling them. I will provide some resources at the end:
- diskutil list
- diskutil unmount disk# (Choose the one that is the USB drive after finding it in diskutil list)
- sudo dd if=/Users/amos/Downloads/ArchLinux.iso of=/dev/disk2 bs=8m
- It will take about 5 minutes to complete. Once it is done type diskutil eject disk#

3) SAN - Now that you got your bootable USB, plug that shit into the computer you want to install Arch Linux on and boot it up
4) FOUR - Enter boot mode, you gotta press f11 or esc or some other key. Just mash whatever key it is to get into boot mode
5) Wow I just noticed I'm typing out the number twice. Anyways choose the Arch Linux iso to boot it up.
6) You should now be at a terminal window. This is the installation screen for Arch!! Completely on the terminal!
7) At this point, since this is just a history of notes for myself for my future self in case I ever do this again.

With that said; future Amos = RTFM
